Inicio Biblioteca Ferramentas Guias Contato
Com a finalidade de facilitar, abaixo fica listada algumas ferramentas onlines que podem ser de grande utilidade para a sua mesa, seja você jogador ou mestre.
Algumas ferramentas têm acesso limitado, sendo necessário pagar para liberar as demais funções da plataforma.

Watabou

Watabou é uma ferramenta online gratuita, voltada para a geração de mapas de cidades, vilarejos, cavernas entre outros recursos

Inkarnate

Inkarnate é uma plataforma online para criação de mapas, com inumeras ferramentas para detalhar ao máximo seu cenário

Owlbear Rodeo

Owlbear Rodeo Roll20 é uma ferramenta gratuita para jogar RPGs de mesa via online, funcionando no próprio navegador

Roll20

Roll20 é uma ferramenta gratuita para jogar RPGs de mesa via online, funcionando no próprio navegador

Firecast

FireCast é um programa para Windows feito para mestres e jogadores de RPG de mesa. Com este sistema instalado no seu PC você poderá jogar partidas online

RollAdvantage

RollAdvantage é um site de ferramentasonline que permite fazer a criação de Tokens para personagens
